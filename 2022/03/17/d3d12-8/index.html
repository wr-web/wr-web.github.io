漫反射（diffuse）
$$
c_{d}=max(L \cdot n,0) \cdot B_{L} \otimes m_{d}
$$

6

环境光（ambient），用于简单模拟其他物体反射过来的间接光照
$$
c_{a}=A_{L} \otimes m_{d}
$$

7

Fresnel equations（菲涅尔等式），Schlick approximation

粗糙度（roughness），镜面瓣（specular lobe）
镜面反射光（specular ）


$$
c_{s}=max(L \cdot n,0) \cdot B_{L} \otimes R_{F}(\alpha_{h}) \frac{m+8}{8}(n \cdot h)^{m} \quad
$$

8

总结了光照模型的概述
$$
LitColor = c_{a}+c_{d}+c_{s}\
LitColor = A_{L} \otimes m_{d}+max(L \cdot n,0) \cdot B_{L} \otimes (m_{d}+R_{F}(\alpha_{h}) \frac{m+8}{8}(n \cdot h)^{m} )\quad\
L:指向光源的向量\
n:表面法线\
h:光向量和观察向量之间的中间向量\
A_{L}:入射环境光量\
B_{L}:入射直射光量\
m_{d}:根据表面反射率反射的入射光量\
L \cdot n :朗伯余弦定理\
\alpha_{h}:中间向量h与光向量L之间的夹角\
R_{F}(\alpha_{h}):根据菲涅尔效应，关于中间向量h所反射到观察者眼中的光量\
m:控制表面粗糙度\
(n \cdot h)^{m}:指定h和n之间\theta_{h}的所有微平面片段的分布情况（所占比例）\
\frac {m+8}{8}:\quad镜面反射过程中，为了模拟能量守恒所采用的归一化因子\
$$

水流的效果只用了一个直接光，那我就改以下那个看看。

// 如果没有把所有可能包含进去会报错，不是很理解成因
float kdtrans(float kd)
{
    if(kd <= 0.0f){
        return 0.4f;
    }else if(kd <= 0.5f){
        return 0.6f;
    }else if(kd <= 1.0f){
        return 1.0f;
    }else{
        return 1.0f;
    }
}

float kstrans(float ks)
{
    if(ks <= 0.0f){
        return 0.0f;
    }else if(ks <= 0.1f){
        return 0.0f;
    }else if(ks <= 0.8f){
        return 0.5f;
    }else if(ks <= 1.0f){
        return 0.8f;
    }else{
        return 0.8f;
    }
}
//修改直接光模型
float ndotl = kdtrans(max(dot(lightVec, normal), 0.0f));
//修改BlinnPhong模型
float roughnessFactor = (m + 8.0f)*kstrans(pow(max(dot(halfVec, normal), 0.0f), m)) / 8.0f;
